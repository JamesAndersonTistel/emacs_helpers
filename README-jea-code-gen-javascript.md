# jea-code-gen for typescript

This is a tool to generate boiler plate code for javascript.

## interactive use
### one time
`jea-code-gen-use-javascript`

### use
`jea-code-gen-prompt`

## dictionaries

enter: `dict,d1,a,1,bee,42.33,see,far`

and it will expand to:

```
let d1 = {
    'a': 1,
    'bee': 42.33,
    'see': 'far'
};
```
